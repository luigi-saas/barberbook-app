# Dev Status

Quick project health check:

1. Run `git status` and summarize uncommitted changes
2. Run `git log --oneline -10` and show recent commits
3. Check `TODO.md` for open tasks
4. List any `.env.local` files that are missing (check apps/app, apps/web, apps/api)
5. Report current branch and whether it's ahead/behind origin

Summarize everything in a short bullet list.
